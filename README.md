# SmartCareer AI

> AI-powered career assistant for resume analysis, interview prep, and job matching

🚀 **Live Demo:** [smartcareer-ai.vercel.app](update it later)

## Tech Stack
- **Frontend:** Next.js 15, TypeScript, Tailwind CSS, shadcn/ui
- **Backend:** Node.js, Express, TypeScript
- **AI:** Google Gemini API
- **Auth:** NextAuth.js
- **Database:** PostgreSQL + Prisma (coming soon)

## Features
- ✅ AI Resume Analyzer with ATS optimization
- ✅ Interview Prep with role-specific questions
- ✅ Job Description Matcher
